# Smart Study Planner with AI Suggestions

# Overview
This project is a Smart Study Planner built using Java. It helps students create an efficient study schedule based on subject difficulty and deadlines, and provides intelligent suggestions to improve productivity.

# Features
- Add subjects with difficulty and deadlines
- Automatically generates study plan (hours/day)
- AI-based suggestions:
  - Focus more on difficult subjects
  - Alerts for urgent deadlines
- User-friendly GUI using Java Swing

# Technologies Used
- Java
- Java Swing (GUI)
- ArrayList (Data Storage)

# How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Krutiaswani/Smart_Study_Planner_Java.git
   Open in VS Code / IntelliJ
2. Open in VS Code / IntelliJ
3. Compile and run:
     - javac Main.java
     - java Main

# How to Use
- Enter subject name
- Enter difficulty (1–5)
- Enter deadline (in days)
- Click "Add Subject"
- View study plan and AI suggestions

# Output
The application opens a GUI window where users can input subjects and view results instantly.
